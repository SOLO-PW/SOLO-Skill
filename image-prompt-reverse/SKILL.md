# Image Prompt Reverse

## 概述

本技能用于分析用户上传的图片，生成精准、全面的AI绘图提示词。支持多模型格式输出，帮助用户在不同AI绘图平台上复现类似效果的图像。

## 核心能力

1. **图片深度分析** - 系统化提取图片中的视觉元素
2. **多模型格式输出** - 适配主流AI绘图模型的提示词格式
3. **质量词推荐** - 自动推荐正向质量词和反向提示词
4. **风格识别** - 准确判断图片的艺术风格

## 工作流程

### 步骤1: 接收图片

确认用户已上传图片。如果用户未上传图片，提示用户提供。

### 步骤2: 深度分析图片

使用**五层分析法**系统化分析图片：

```
第一层: 整体印象 - 风格、氛围、情绪基调
第二层: 主体内容 - 人物/物体特征、数量、位置
第三层: 环境背景 - 场景、地点、背景元素
第四层: 技术细节 - 光照、视角、构图、景深
第五层: 质量特征 - 分辨率、细节程度、画面质感
```

详细分析方法参见 [references/analysis-method.md](references/analysis-method.md)。

### 步骤3: 确定输出模型

询问用户需要哪些模型的提示词格式，或默认输出以下常用模型：

| 模型 | 特点 |
|------|------|
| Stable Diffusion | 权重语法，需质量词 |
| Midjourney | 参数语法，自然描述 |
| DALL-E 3 | 纯自然语言描述 |
| Flux | 自然语言+风格词 |
| NijiJourney | 动漫风格专用 |

详细格式规范参见 [references/models.md](references/models.md)。

### 步骤4: 生成提示词

按以下结构组织提示词：

```
[主体描述] + [特征细节] + [环境背景] + [光照氛围] + [风格] + [质量词]
```

### 步骤5: 输出质量词

自动生成配套的质量增强词和反向提示词。参见 [references/quality-words.md](references/quality-words.md)。

## 提示词编写原则

### 精准描述

- **具体优于抽象**: 使用"flowing silver hair"而非"nice hair"
- **可量化优于模糊**: 使用"35mm lens"而非"normal lens"
- **专业术语**: 使用"chiaroscuro lighting"而非"dramatic lighting"

### 权重分配

| 元素类型 | 建议权重 |
|----------|----------|
| 核心主体 | 1.2-1.4 |
| 风格词 | 1.0-1.2 |
| 质量词 | 1.1-1.3 |
| 背景元素 | 0.8-1.0 |

### 长度建议

| 模型 | 推荐长度 |
|------|----------|
| Stable Diffusion | 50-150 tokens |
| Midjourney | 30-80 words |
| DALL-E 3 | 100-400 words |
| Flux | 50-150 words |
