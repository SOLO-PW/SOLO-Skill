## 识别原则

### 核心原则

1. **特征组合匹配** - 不依赖单一特征，通过发型+发色+眼睛+服装+配饰的组合判断
2. **作品风格关联** - 识别画风后缩小角色范围
3. **标志性元素优先** - 优先匹配最具辨识度的特征（如独特配饰、发型）
4. **不确定时描述特征** - 无法确定具体角色时，详细描述可见特征而非猜测

### 识别置信度

| 置信度 | 判断标准 | 输出方式 |
|--------|----------|----------|
| 高 | 3个以上标志性特征匹配 | 直接说出角色名和作品 |
| 中 | 2个特征匹配，风格吻合 | "疑似为XX作品中的XX角色" |
| 低 | 仅有1个特征匹配 | 不猜测角色名，详细描述特征 |

## 角色识别流程

```
Step 1: 识别作品风格
  ├─ 日式动漫 → 进入Step 2
  ├─ 美式卡通 → 进入Step 2
  ├─ 国漫/韩漫 → 进入Step 2
  └─ 无法判断风格 → 直接描述特征

Step 2: 提取角色特征
  ├─ 发型（长度、颜色、形状、标志性造型）
  ├─ 眼睛（颜色、形状、特殊效果）
  ├─ 服装（校服、制服、战甲、日常装）
  ├─ 配饰（发饰、武器、道具、耳环等）
  └─ 其他（动物伙伴、纹身、特殊标记）

Step 3: 特征匹配
  ├─ 高置信度匹配 → 输出角色名+作品
  ├─ 中置信度匹配 → 输出疑似角色+特征描述
  └─ 低置信度/无匹配 → 详细描述所有可见特征

Step 4: 生成角色描述
  └─ 按角色描述框架组织输出
```

## 特征匹配方法

### 高辨识度特征（权重最高）

以下特征最具辨识度，优先匹配：

| 特征类型 | 说明 | 示例 |
|----------|------|------|
| 独特发型 | 非常规造型、标志性发色 | 刺猬头、双马尾、渐变发色 |
| 标志性配饰 | 角色专属物品 | 护额、面具、特定武器 |
| 独特服装 | 角色标志性穿着 | 校服款式、战斗服 |
| 特殊标记 | 身体上的独特标记 | 纹身、伤疤、特殊瞳色 |
| 动物伙伴 | 角色常伴的动物 | 小龙、猫、精灵 |

### 中辨识度特征

| 特征类型 | 说明 |
|----------|------|
| 发色 | 常见色（黑/棕/金）辨识度低，罕见色（粉/蓝/紫/红）辨识度高 |
| 瞳色 | 特殊瞳色（异色瞳、金色、红色）辨识度高 |
| 体型 | 特殊体型（极高、极矮、特殊比例） |
| 表情习惯 | 角色标志性表情 |

### 低辨识度特征

| 特征类型 | 说明 |
|----------|------|
| 常规服装 | 普通校服、日常装 |
| 常规发型 | 常见短发、长发 |
| 常规发色 | 黑色、棕色 |

## 角色描述框架

### 识别成功时

```
角色名 (作品名)
├─ 基本信息: 性别、年龄外观、身高体型
├─ 外貌特征: 发型发色、眼睛颜色、面部特征
├─ 服装造型: 标志性服装、配饰、武器
├─ 表情姿态: 角色标志性表情或姿态
└─ 风格参考: 作品画风、官方设定图风格
```

### 识别失败时

```
未知角色
├─ 基本信息: 性别、年龄外观、身高体型
├─ 外貌特征: 发型发色（详细）、眼睛颜色（详细）、面部特征
├─ 服装造型: 服装类型、颜色、配饰、武器（详细描述）
├─ 标志性元素: 最独特的1-2个视觉特征
├─ 画风推测: 可能的作品风格或地区风格
└─ 参考建议: 可参考的类似角色或风格
```

## 角色提示词模板

### 已识别角色模板

**Stable Diffusion:**
```
(角色名:1.3), (作品名 style:1.2), [角色标志性特征描述], [服装描述], [表情姿态], [背景], [画风描述], masterpiece, best quality, highly detailed

示例:
(Anya Forger:1.3), (Spy x Family style:1.2), (pink short hair:1.2), (green eyes:1.1), (hair ornaments:1.1), wearing school uniform, surprised expression, holding stuffed penguin, classroom background, anime illustration style, masterpiece, best quality
```

**Midjourney:**
```
角色名 from 作品名, [标志性特征], [服装], [表情姿态], [背景], [画风] --ar 比例 --niji 版本 --style 风格

示例:
Anya Forger from Spy x Family, pink short hair with hair ornaments, green eyes, school uniform, surprised expression, classroom background, anime illustration --ar 2:3 --niji 6 --style cute
```

**DALL-E 3:**
```
Anya Forager from the anime Spy x Family. A young girl with short pink hair and green eyes, wearing hair horn ornaments. She is wearing a school uniform and has a surprised expression. Anime illustration style with soft colors.
```

### 未识别角色模板

当无法识别具体角色时，使用详细特征描述：

```
[性别] anime character with [详细发型描述], [详细眼睛描述], wearing [详细服装描述], [配饰描述], [表情姿态], [背景], [画风推测], highly detailed, masterpiece

示例:
Male anime character with spiky silver hair and black mask covering lower face, wearing dark blue outfit with metal forehead protector tilted to cover one eye, holding a kunai, serious expression, forest background, ninja anime style, highly detailed, masterpiece
```

### 角色Cosplay/真人风格模板

当图片是真人cosplay或3D写实风格的角色时：

```
cosplay photography of [角色名] from [作品名], [角色特征描述], realistic portrayal, [服装材质描述], [摄影风格], [质量词]

示例:
cosplay photography of Levi Ackerman from Attack on Titan, short black hair combed back, gray-blue eyes, white scarf, Survey Corps uniform with ODM gear, realistic portrayal, studio lighting, professional cosplay photography, 8k, highly detailed
```

## 角色识别注意事项

### 同一角色不同形态

许多角色有多种形态，需要识别当前形态：

| 形态类型 | 示例 | 处理方式 |
|----------|------|----------|
| 变身形态 | 超级赛亚人、九尾模式 | 描述当前形态 |
| 年龄变化 | 小时候vs长大后 | 描述当前年龄外观 |
| 服装变化 | 日常装vs战斗装 | 描述当前服装 |
| 状态变化 | 觉醒后、黑化后 | 描述当前状态 |

### 原创角色处理

当图片是原创角色（非已知作品角色）时：

1. **不要强行匹配**已知角色
2. 详细描述所有可见特征
3. 说明"疑似原创角色"
4. 提供风格参考建议

### AI生成角色处理

当图片是AI生成的角色时：

1. 识别画风（哪个AI模型或哪种风格）
2. 描述角色的设计特征
3. 说明"AI生成角色"
4. 提供风格关键词以便复现
