## 分析前准备

### 图片质量评估

分析前首先评估图片质量：

- [ ] **分辨率**: 图片是否足够清晰以辨认细节
- [ ] **光照**: 是否存在过暗或过曝导致细节丢失
- [ ] **遮挡**: 是否有物体遮挡关键特征
- [ ] **角度**: 角度是否影响特征识别

### 图片类型识别

确定图片类型以调整分析重点：

| 类型 | 分析重点 | 注意事项 |
|------|----------|----------|
| 人物肖像 | 面部特征、表情、服装 | 注意性别识别准确性 |
| 人物全身 | 姿态、服装、身体比例 | 注意服装细节 |
| 风景/场景 | 环境、氛围、构图 | 注意空间层次 |
| 静物 | 物体特征、材质、光影 | 注意细节纹理 |
| 抽象/概念 | 色彩、形状、情绪 | 注意艺术表达 |
| 动漫/插画 | 风格特征、线条、色彩 | 注意风格一致性 |

## 分层检查清单

### 第一层：整体印象

**必检项目**:
- [ ] 图片整体氛围（明亮/昏暗/神秘/温馨等）
- [ ] 主导色调（暖色/冷色/黑白/彩色）
- [ ] 艺术风格（写实/动漫/油画/水彩/3D等）
- [ ] 情绪基调（快乐/悲伤/紧张/平静等）
- [ ] 画面焦点位置

**验证问题**:
- 这张图片给你的第一印象是什么？
- 如果要用三个词描述整体感觉，会是什么？

### 第二层：主体内容

#### 人物主体检查清单

**基础信息**:
- [ ] 人物数量（单人/多人/群体）
- [ ] 人物位置（前景/中景/背景）
- [ ] 人物占比（特写/半身/全身/远景）

**性别识别** (参见 gender-identification.md):
- [ ] 服装是否有明显性别特征
- [ ] 发型长度和款式
- [ ] 面部特征（写实风格）
- [ ] 配饰类型
- [ ] 身体轮廓（若可见）
- [ ] **交叉验证**: 至少2个特征一致才能确定性别

**年龄判断**:
- [ ] 面部特征（皱纹、皮肤状态）
- [ ] 发型（白发、脱发等）
- [ ] 服装风格
- [ ] 姿态和肢体语言

**种族/民族特征** (谨慎处理):
- [ ] 肤色
- [ ] 面部轮廓特征
- [ ] 发型特征
- [ ] 服装文化元素
- [ ] **注意**: 避免刻板印象，客观描述可见特征

**外貌细节**:
- [ ] 发型（长度、颜色、款式、质感）
- [ ] 眼睛（颜色、形状、表情）
- [ ] 面部特征（轮廓、五官特点）
- [ ] 表情（情绪状态）
- [ ] 特殊标记（痣、疤痕、纹身等）

**服装细节**:
- [ ] 服装类型（上衣/下装/外套/鞋子）
- [ ] 颜色（主色、辅色、配色方案）
- [ ] 材质（棉/丝/皮革/金属等）
- [ ] 款式（正式/休闲/传统/现代）
- [ ] 装饰（图案、配饰、徽章等）
- [ ] 状态（新旧、整洁/破损）

**姿态动作**:
- [ ] 基本姿态（站/坐/躺/走/跑等）
- [ ] 身体朝向（正面/侧面/背面/四分之三）
- [ ] 手势（手部位置、动作）
- [ ] 视线方向
- [ ] 动态感（静止/运动）

#### 物体主体检查清单

- [ ] 物体名称和类型
- [ ] 数量
- [ ] 大小比例
- [ ] 形状特征
- [ ] 颜色
- [ ] 材质
- [ ] 状态（新旧、完好/破损）
- [ ] 位置关系

#### 场景主体检查清单

- [ ] 场景类型（室内/室外/自然/城市/幻想）
- [ ] 具体地点（房间类型、地貌特征）
- [ ] 主要元素
- [ ] 空间层次（前景/中景/背景）

### 第三层：环境背景

**空间环境**:
- [ ] 室内/室外
- [ ] 封闭/开放空间
- [ ] 空间大小（宽敞/狭窄）
- [ ] 建筑元素（墙壁、地板、天花板）
- [ ] 自然元素（天空、地面、植被、水体）

**时间信息**:
- [ ] 时段（黎明/白天/黄昏/夜晚）
- [ ] 光线方向判断时间
- [ ] 天空状态（晴朗/多云/星空）

**天气条件** (室外场景):
- [ ] 天气状况（晴/阴/雨/雪/雾）
- [ ] 能见度
- [ ] 特殊天气效果（彩虹、闪电等）

**背景元素**:
- [ ] 背景复杂度（简单/复杂）
- [ ] 背景虚化程度
- [ ] 背景中的显著物体
- [ ] 背景与主体的关系

### 第四层：技术细节

**光照分析**:
- [ ] 光源类型（自然光/人造光/混合光）
- [ ] 光源方向（正面/侧面/背面/顶光/底光）
- [ ] 光线质量（硬光/软光）
- [ ] 光色温（暖色/冷色/中性）
- [ ] 特殊光效（体积光、霓虹、反射等）
- [ ] 阴影特征（方向、硬度、长度）

**色彩分析**:
- [ ] 主色调
- [ ] 配色方案（单色/互补/类似色等）
- [ ] 饱和度（高饱和/低饱和/黑白）
- [ ] 对比度（高对比/低对比）
- [ ] 色彩情绪

**视角构图**:
- [ ] 拍摄角度（平视/俯视/仰视/鸟瞰/虫视）
- [ ] 镜头类型（广角/标准/长焦/微距）
- [ ] 构图方式（中心/三分法/对称/引导线等）
- [ ] 画幅比例暗示

**景深效果**:
- [ ] 景深类型（浅景深/深景深）
- [ ] 焦点位置
- [ ] 虚化质量（圆形光斑/多边形光斑）

**画质特征**:
- [ ] 清晰度（锐利/柔和/模糊）
- [ ] 噪点情况
- [ ] 动态模糊
- [ ] 特殊效果（颗粒、光晕等）

### 第五层：质量特征

**细节程度**:
- [ ] 纹理细节（皮肤、织物、材质）
- [ ] 边缘锐度
- [ ] 微小元素识别

**艺术处理**:
- [ ] 笔触可见性
- [ ] 数字处理痕迹
- [ ] 滤镜效果
- [ ] 特殊艺术处理

**技术缺陷** (如有):
- [ ] 过曝/欠曝区域
- [ ] 色偏
- [ ] 畸变
- [ ] 压缩痕迹

## 验证机制

### 交叉验证规则

**性别验证**:
```
必须满足: 服装特征 + (发型特征 或 面部特征)
或: 明确的性别化服装 (如裙子/西装)
```

**风格验证**:
```
必须满足: 至少2个风格特征一致
例如: 大眼睛 + 简化面部 = 动漫风格
```

**光照验证**:
```
必须满足: 阴影方向 + 高光位置 + 光源方向 一致
```

### 不确定性标注

当以下情况出现时，必须标注不确定性：

| 情况 | 标注方式 |
|------|----------|
| 性别不确定 | "性别特征不明显" / "中性风格" |
| 年龄不确定 | "年龄难以判断" / "年轻至中年" |
| 风格混合 | "融合多种风格" / "介于X与Y之间" |
| 细节不清 | "细节模糊，可能为..." |
| 遮挡严重 | "部分被遮挡，可见..." |

### 自检问题

完成分析后，自问以下问题：

1. **准确性检查**:
   - 我确定的性别是否有至少2个特征支持？
   - 我描述的特征是否都能在图片中看到？
   - 我是否避免了主观臆测？

2. **完整性检查**:
   - 是否涵盖了所有五层分析？
   - 主体、环境、技术细节是否都有描述？
   - 是否遗漏了显著特征？

3. **一致性检查**:
   - 光照描述是否与阴影一致？
   - 视角描述是否与画面透视一致？
   - 风格描述是否与视觉特征一致？

4. **实用性检查**:
   - 提示词是否足够详细以复现效果？
   - 是否包含了关键的质量词？
   - 反向提示词是否针对性强？

## 常见错误预防

### 性别误认预防

**高风险场景**:
- 动漫风格人物
- 中性服装
- 背影/侧影
- 非传统性别表达

**预防措施**:
1. 优先查看服装
2. 多重特征交叉验证
3. 不确定时使用中性描述
4. 避免基于单一特征判断

### 风格误判预防

**易混淆风格**:
- 3D渲染 vs 照片写实
- 油画 vs 数字绘画
- 动漫 vs 卡通

**预防措施**:
1. 查看纹理细节（笔触、像素、光滑度）
2. 检查边缘处理方式
3. 观察光影的物理准确性
4. 注意色彩的自然度

### 细节遗漏预防

**易遗漏细节**:
- 微小配饰
- 背景中的文字/标志
- 特殊材质效果
- 情绪微表情

**预防措施**:
1. 按检查清单逐项核对
2. 放大查看细节区域
3. 注意边缘和角落
4. 检查反射和阴影中的信息

## 质量评估标准

### 分析质量等级

| 等级 | 标准 | 要求 |
|------|------|------|
| A级 | 精准全面 | 所有检查项完成，无错误，描述详尽 |
| B级 | 准确完整 | 主要检查项完成，无重大错误 |
| C级 | 基本合格 | 关键信息正确，有少量遗漏 |
| D级 | 需要改进 | 有明显错误或重大遗漏 |

### 提示词质量评估

**优秀提示词特征**:
- ✅ 具体、可量化的描述
- ✅ 按重要性排序
- ✅ 包含风格和质量词
- ✅ 反向提示词针对性强
- ✅ 适合目标模型格式

**劣质提示词特征**:
- ❌ 模糊、主观描述
- ❌ 重要特征遗漏
- ❌ 风格和质量词缺失
- ❌ 反向提示词过于通用
- ❌ 格式不符合模型要求
